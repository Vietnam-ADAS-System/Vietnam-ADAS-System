# Frontend - Vietnam ADAS System

React-based frontend for the Advanced Driver Assistance System.

## Structure

- `public/` - Static assets (favicon, logo, etc.)
- `src/pages/` - Main pages (Dashboard, Detection, LaneDetection, TrafficSign)
- `src/components/` - Reusable components (VideoPlayer, WarningPanel, DetectionTable, TrafficSignPanel)
- `src/services/` - API services for communicating with Node.js backend
- `src/assets/` - Images, icons, and other assets

## Installation

```bash
npm install
npm start
```
